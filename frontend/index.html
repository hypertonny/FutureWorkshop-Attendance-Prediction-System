Technical Logs

Hyperparameter Tuning ▼
Used GridSearchCV with 5-fold cross validation. Penalty: l2, C: 1.0, Solver: lbfgs. Optimized for F1-macro to handle slight class imbalance.
Feature Engineering ▼
Created 'Academic Weight' feature by combining semester load and exam proximity. Applied One-Hot Encoding to categorical topics.
Scaling Strategy ▼
StandardScaler applied to registration counts and duration to ensure convergence in the Logistic Regression model.
